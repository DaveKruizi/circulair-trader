⚙️ GitHub toegangstoken

Om posities te registreren stuurt het dashboard een opdracht naar GitHub. Daarvoor heb je eenmalig een persoonlijk toegangstoken (PAT) nodig.

Aanmaken in 3 stappen:
1. Ga naar github.com/settings/tokens/new
2. Geef het een naam (bijv. "Lego dashboard") en vink workflow aan
3. Klik Generate token en plak het hieronder

Wordt alleen lokaal opgeslagen in je browser. Nooit verstuurd naar anderen.
